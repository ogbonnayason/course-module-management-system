# Course Module Management System

## 📖 Overview

A Java-based Course Module Management System web application for managing course modules, student enrollments, and progress tracking.
Built using OOP principles, Maven following MVC architecture, and design patterns (DAO, Singleton) with JSP/Servlet, and MongoDB.

## ❗ Problem

Manual course enrollment systems are inefficient, error-prone, and difficult to manage.

## 🚀 Features

* User authentication (login & signup)
* Module management (CRUD)
* Student enrollment system
* Progress tracking dashboard

## 🧱 Tech Stack

* Java (Maven)
* JSP & Servlets
* MongoDB Atlas
* Apache Tomcat

## 🧠 Concepts Used

* OOP (Encapsulation, Inheritance, Polymorphism)
* MVC Architecture
* Design Patterns (DAO, Singleton)

------

Deploy on Tomcat and configure MongoDB connection.

------

## 📁 Structure

* `model/` – Data models
* `dao/` – Database logic
* `servlet/` – Controllers
* `jsp/` – Views

## 🎓 Skills

Java web development, OOP design, MVC, database integration, testing.
